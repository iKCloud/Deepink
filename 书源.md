

## 厚墨
      极简设计，纯粹阅读。无广告，无推送。
      支持 epub  mobi azw3 格式文件。
      支持添加 JsonPath 与 CSS Query 指定规则的书源。
      支持付费订阅使用云端相关功能。
      群组：订阅用户可以通过发现>设置>内测群组申请进群。
      格式：内建一种基于 Markdown 的图书格式. mp ， epub  mobi  azw3 将会弃掉格式转为. mp 。
      内建格式聚焦于存储，而格式渲染由阅读器统一完成。
